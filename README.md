# daz-dotfiles

My personal dotfiles and XFCE4 setup.

## Included

- XFCE4 configs
- GTK themes
- Fonts, icons
- Neofetch setup
- Full install script: `install.sh`

## Installation

```bash
git clone https://github.com/dazui-py/daz-dotfiles.git ~/daz-dotfiles
cd ~/daz-dotfiles
chmod +x install.sh
./install.sh
